#### 실패 결과에서 원인을 거꾸로 추적
이러한 실험은 여러 GPU에 재현되는지 확인해야 했는데, 결과적으로는 각각의 GPU가 모델 조정 결과를 달라진 실제 데이터와 비교하는 과정에서 큰 차이를 보였습니다. 예를 들어, AGPU는 50만 값을 기준으로 성능이 크게 좋아졌지만, BGPU는 5000만 값에 가까운 경우에 최적의 결과가 나오고 있었습니다.

### GPU 별 재현 조건과 실제 데이터 간의 차이
이는 주로 모델 학습 데이터와 실제 사용 환경 사이의 차이 때문일 것으로 보입니다. 예를 들어, 실제 사용 환경은 서버실 온도가 42도까지 올라가는 경우도 있고, 다른 GPU는 상대적으로 낮은 온도를 유지하고 있었습니다.

#### 실패 결과에서 원인을 거꾸로 추적
